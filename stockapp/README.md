# Stock Trader Django Application

A Django web application for viewing stock analysis data generated by the nifty100_retest.py script.

## Features

- User authentication (login/logout)
- Display stocks from different analysis categories:
  - Retest stocks (upward/downward)
  - Bullish stocks
  - Bearish stocks
- Interactive day chart for selected stocks

## Setup

1. Install dependencies:
   ```bash
   pip install django yfinance pandas numpy
   ```

2. Run migrations:
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

3. Create a superuser:
   ```bash
   python manage.py createsuperuser
   ```

4. Update stock data:
   ```bash
   python manage.py update_stocks
   ```

5. Run the development server:
   ```bash
   python manage.py runserver
   ```

## Usage

1. Navigate to `/login/` and log in with your superuser credentials
2. After login, you'll see the dashboard with a list of stocks on the left
3. Click on any stock to view its intraday chart on the right
4. Use the logout link to sign out

## Models

- **RetestStock**: Stores stocks that are in retest phase
- **BullishStock**: Stores stocks with bullish indicators
- **BearishStock**: Stores stocks with bearish indicators

## Management Commands

- `update_stocks`: Fetches latest stock data from nifty100_retest.py and updates the database

## API Endpoints

- `GET /api/stock-chart/<symbol>/`: Returns intraday chart data for a stock symbol
